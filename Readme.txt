# Alert CO

## Project setup
```
npm install
```

### Start Production with PM2 
```
pm2 start ecosystem.config.js --env production
```

Apps    : Alert CO

- .vscode -> vscode settings
- config -> Folder Configurasi pembacaan dan validasi file .env
- controllers -> gateway / route layer
- keys -> key SSL
- pluggins -> fastify Plugins
- services -> Bispro
